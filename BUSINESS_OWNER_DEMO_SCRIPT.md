# Skyrich Battery Türkiye — Business Owner Demo Script

## Demo Flow (5–7 minutes)

### 1. Homepage (1 min)
- Open `https://www.skyrichbattery.com.tr`
- Show hero section with dynamic title and CTA buttons
- Mention: "Hero text and buttons can be changed from the admin panel"
- Scroll to "Neden Skyrich?" section — show clean, safe copy (no fake claims)
- Show product grid with approved SKUs only

### 2. Product Catalog (1 min)
- Navigate to "Ürünler"
- Show product cards with images, names, and model codes
- Click a product → show detail page
- Show technical specs (voltage, capacity, CCA, weight)
- Note: "Specs are entered from admin; unverified ones show 'Doğrulanacak'"

### 3. Battery Finder / Help CTA (1 min)
- Navigate to "Akü Bulucu"
- Show vehicle make/model/year search
- Or click WhatsApp CTA: "Uyumlu Aküyü Birlikte Bulalım"
- Explain: "Customers can message via WhatsApp to find the right battery"

### 4. Contact Page (30 sec)
- Navigate to "İletişim"
- Show WhatsApp, phone, email, and address cards
- Explain: "All contact info is managed from the admin panel"

### 5. Admin Login (30 sec)
- Go to `https://www.skyrichbattery.com.tr/admin`
- Log in with admin password
- Show admin dashboard with stats cards

### 6. Live Edit Demo (2 min)
- Go to **Hero Ayarları** tab
- Change the hero title → click Kaydet
- Open homepage in a new tab → show updated hero text
- Go back to admin
- Go to **Popup** tab
- Edit popup content → save
- Open homepage → show updated popup

### 7. Closing Notes (1 min)
- "This is a catalog and contact website, not e-commerce."
- "No prices, stock, or checkout — customers contact you via WhatsApp."
- "You can update all text, images, products, and contact info yourself."
- "Future updates: add new products, change banners, update specs — all from admin."
- "The site is SEO-ready with sitemap, robots, and structured data."

## Key Messages

- **No fake claims**: Every spec and marketing claim is either verified or shows "Doğrulanacak"
- **Admin-managed**: Hero, popup, banners, products, contact info, and page text are all editable
- **WhatsApp-first**: Primary customer contact is WhatsApp, not a fake form
- **SKU whitelist**: Only approved 12 Skyrich SKUs are shown publicly
- **Secure**: Admin panel is protected by password and bearer token; no public admin link

## Common Questions

**Q: Can we add prices later?**
A: Yes, but that requires e-commerce features. For now, the focus is catalog + contact.

**Q: Can we change the design?**
A: The dark premium industrial style is locked. Content (text, images, products) is fully editable.

**Q: How do we add a new product?**
A: Go to Admin → Aküler → Yeni Ekle. Fill model code, name, image URL, specs. Mark active when ready.

**Q: Is the site mobile-friendly?**
A: Yes. Test it on your phone — the menu, product grid, and contact cards all adapt.
